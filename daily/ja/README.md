# 日本語 daily notes

日々の運用メモはここに置きます。

この場所は、完成したドキュメントを書くためではなく、発見を早く安全に残すための場所です。再利用できる知見になったら `docs/en/` に昇格します。

## 書いてよいもの

- 何が起きたか
- 何が誤解しやすかったか
- どの層が壊れていたか
- 次回どこを見るべきか
- stable docs に昇格する価値があるか

## 書かないもの

- token、cookie、credential、secret
- 実IP、実ドメイン、実メールアドレス
- channel ID
- 個人用の agent 名
- raw production log
- private script や wrapper の中身
